【孝親首選】股票大字管家：專為長輩設計的台美股記帳本 (免月費、大字體)

老兵 一月 28, 2026

您家裡的長輩是否也有這樣的困擾？

  • ❌ 券商 App 的字太小，老花眼看得很吃力。
  • ❌ 功能太複雜，K線圖、技術指標一大堆，其實只想看「賺多少」。
  • ❌ 有買美股 (如 NVDA, TSLA)，但還要自己按計算機乘匯率，才知道台幣價值。

身為一個懂程式的晚輩，我決定自己動手，寫一套「專門給長輩用的股票儀表板」！經過多次改良，這套工具現在字體超大、介面乾淨，而且台股美股都能通吃。

不用懂複雜的 GitHub，我已經打包好放在 Google Drive，今天就免費分享給大家！

🌟 為什麼長輩會愛上這個工具？

1. 字體真的超級大！

我特別調整了介面設計，不管是標題、數字還是表格，字體都比一般網頁大 30% 以上。在大螢幕上看更舒服，完全不用瞇著眼睛找數字。

2. 自動換算台幣，資產一目了然

系統會自動連線抓取即時美金匯率。就算長輩買的是輝達 (NVDA) 或蘋果 (AAPL)，系統也會自動幫忙乘上匯率，直接顯示「約等於多少台幣」，不用再心算了。

3. 紅綠分明，只看重點

首頁沒有複雜的線圖，只有四張大卡片：成本、市值、總損益、今日漲跌。賺錢顯示紅色，賠錢顯示綠色，直覺又簡單。

(此處建議放上您 V26 版本的首頁截圖)

📖 超簡單安裝與使用教學

事前準備 (只需做一次)

因為這是一個 Python 小程式，電腦裡需要先有 Python 環境。如果長輩的電腦沒有，請晚輩幫忙安裝一下：

  1. 下載並安裝 Python (點此前往官網)。
  2. 安裝完成後，打開電腦的「命令提示字元 (CMD)」，輸入以下指令安裝需要的套件：
    pip install streamlit pandas yfinance plotly

🚀 如何開始使用？

環境設定好後，以後長輩要看股票，只需要做這件事：

  1. 下載我提供的壓縮檔並解壓縮。
  2. 在桌面上找到 StockManager.bat 這個檔案。
  3. 點兩下，網頁就會自動跳出來了！

✍️ 如何記帳？

程式介面分為幾個簡單的分頁：

  • 📊 庫存總覽：一打開就看得到，這裡顯示所有持股的現價與損益。
  • ✍️ 記一筆：
    • 買進：輸入代號 (如 2330 或 NVDA)、股數、價格。台股單位預設為「張」，美股為「股」。
    • 領息：配息入帳時，來這裡選「領息」，輸入金額，系統會幫您統計落袋多少錢。
  • 📜 歷史獲利：這裡專門看「已經賣掉賺的錢」和「領到的股息」，是真正的獲利！
  • 🔍 查配息：輸入代號，幫您查最近 5 年配了多少錢，方便補登紀錄。

💡 結語

科技應該是要帶來便利的，而不是製造門檻。透過這個小工具，希望能讓家裡的長輩在投資路上更輕鬆、看得更清楚。
